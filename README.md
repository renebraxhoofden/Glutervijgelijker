# GlutenvrijGelijker.nl – V7 Final

## Wat is dit?
Dit is de finale multi-file websiteversie met:
- jouw logo verwerkt
- centrale affiliate-links in `assets/js/affiliate-links.js`
- duidelijke SEO-head per pagina
- robots.txt, sitemap.xml en llms.txt
- GitHub Pages workflow

## Wat pas je later het vaakst aan?
- `assets/js/affiliate-links.js` → affiliate-url's
- `contact.html` → mailadres / tekst
- `privacy.html`, `cookiebeleid.html`, `disclaimer.html` → juridische teksten
- `aanbiedingen.html` → teksten / kaarten / producten

## Centrale affiliate-opzet
De knop-url’s worden gevuld met `data-affiliate-key` + `assets/js/affiliate-links.js`.
Dat betekent: inhoud blijft zichtbaar, maar jij beheert url’s centraal.
